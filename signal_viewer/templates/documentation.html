Signal Viewer User Guide

Navigation

File Selection Bar

The topmost bar provides four cascading type-to-filter dropdowns for navigating to a specific HDF5 file:

  • Root: Select the data root directory (auto-selected when only one is configured)
  • Serial: Select a test serial or unit identifier (populated after root selection)
  • Step: Choose a test step (populated after serial selection)
  • File: Select a specific HDF5 file with its size shown in parentheses (populated after step selection)

All four dropdowns support type-to-filter: start typing to narrow the list, use arrow keys to navigate, and Enter to select. Session state (selected root, serial, step, and file) is persisted in sessionStorage so the selection survives page refreshes and tab navigation.

Favourite Heart

A blue heart icon appears next to the File dropdown once a file is selected. Click it to toggle the file as a favourite. Favourited files are stored per user and can be used to filter the file selection via the File Lists combobox.

File Lists Filter

The File Lists combobox at the far right of the file selection bar filters which files are visible in the cascade. Options include:

  • All files (default): No filtering; all files are shown at every cascade level.
  • My favs: Shows only files marked as favourites. The Root, Serial, and Step dropdowns are pruned to only display entries that lead to at least one favourited file.
  • <List name>: Shows only files belonging to the selected user list. Dropdowns are pruned the same way.

Changing the File Lists selection resets all cascade dropdowns and clears the current page content.

Sidebar

The left sidebar displays available signals from the selected file, grouped by batch (e.g. GROUP_T0, GROUP_T1). Click a batch header to expand or collapse it. Click a signal name to load it into the current page. Signals are sorted alphabetically within each batch. Underscores in signal names are replaced with spaces for readability, and a tooltip shows the original name on hover.

The sidebar is resizable: drag the right edge to adjust its width. Both the sidebar width and scroll position are persisted across page navigation.

Tab Navigation

The navigation bar below the file selector links to six sections:

  • Viewer: Interactive signal plotting with adaptive zoom
  • Analysis: Statistical analysis, histograms, and rainflow cycle counting
  • Comparison: Side-by-side comparison of two signals
  • Notes: Private and public notes attached to files
  • Lists: Create and manage named file lists
  • Docs: This documentation page

Viewer

Display Modes

The viewer supports two layout modes, selectable from the Layout dropdown:

  • Same Axis (Overlay): All signals share one set of axes. When a single signal is loaded the y-axis shows its units; with multiple signals it shows "Value".
  • Subplots: Each signal gets its own subplot with an independent y-axis labelled with the signal's units. All subplots share the same time range.

Switching layout mode while zoomed automatically reloads the full overview data so the signal is always displayed correctly.

Time Modes

The Time dropdown switches between two x-axis representations:

  • Absolute: Time axis shows wall-clock timestamps in h:m:s format derived from the epoch-ms values stored in the HDF5 file.
  • Relative: Time axis shows elapsed time from the earliest signal start as H:MM:SS (or H:MM:SS.mmm when zoomed to sub-second precision). The origin is the smallest tStart across all loaded signals.

Both modes work correctly with adaptive zoom — zooming in while in relative mode fetches the correct epoch-ms window from the server and converts back to relative seconds for display.

Signal Cards

The collapsible SIGNAL CARDS section below the controls shows a card for each loaded signal with:

  • Batch name and signal name (colour-coded to match the plot trace)
  • Live hover readout: value, timestamp, and estimated full-signal sample index
  • Point count ratio (displayed points / total samples)
  • Remove button (×) to unload the signal

Click the SIGNAL CARDS label to collapse or expand the cards section.

Adaptive Zoom

The viewer uses an adaptive zoom system that re-fetches data from the server at higher resolution whenever you zoom in:

  • Zoom: Click and drag to select a region. After a short debounce, the server returns up to 8,000 points for just the visible window, giving pixel-level detail at any zoom depth.
  • Pan: Hold shift and drag to pan across the signal.
  • Reset zoom: Three ways to return to the full overview — the house icon (⌂) in the toolbar, the "Reset Zoom" button in the header, or double-clicking the plot.

The server uses a two-pass MinMax-LTTB downsampling algorithm that preserves visual shape while keeping the point count low. Removing a signal while zoomed preserves the current zoom level.

Hover Crosshair

In subplot mode, hovering over any subplot draws a synchronised vertical crosshair across all subplots and updates every signal card with the interpolated value at that time.

Signal Limit

Up to 5 signals can be loaded simultaneously to maintain performance. Remove signals as needed to load new ones.

Analysis

Click a signal in the sidebar to run a full analysis. The page works with one signal at a time.

Statistics

Descriptive statistics for the selected signal:

  • Descriptive Stats: Min, max, mean, standard deviation, median
  • Distribution: Histogram showing value distribution
  • Rainflow Cycle Counting: ASTM E1049 compliant cycle detection

Rainflow Cycle Counting

Identifies stress cycles using the four-point rainflow algorithm:

  • Counts full and half cycles with range calculations
  • Percentile table showing range distribution
  • Range histogram with configurable bin count
  • Exceedance curves (range, cycle max, cycle min) for fatigue analysis

Value Histogram

Visualizes the distribution of the selected signal's values:

  • Automatically determines appropriate bin sizing
  • Shows frequency of values in each bin

Comparison

Click signals in the sidebar to assign them: first click = Signal A, second click = Signal B. Four comparison views are generated automatically:

Overlay Comparison

Both signals on the same time axis to compare waveforms directly. Useful for comparing signals from different test conditions, identifying differences in amplitude and phase, and visual alignment of features.

X vs Y Correlation

Scatter plot showing the relationship between two signals. The x-axis shows Signal A values, the y-axis shows Signal B values. Reveals nonlinear relationships and hysteresis loops.

Combined Histogram

Overlay histograms of two signals for distribution comparison: side-by-side visualization of value ranges and differences in statistical properties.

Cross-Correlation

Measures similarity and time alignment between signals. Detects time lags, quantifies correlation strength (-1 to +1), and is useful for signal synchronization analysis.

Notes

The Notes page (accessible via the Notes tab) provides per-file annotation with two types of notes, displayed in a two-column layout:

Private Notes (left column)

A single private note per file per user. Only visible to the author. The text area is fixed-height (15 lines) with a scroll bar for longer content. Click Save to persist; the save date is shown to the right of the button.

Public Notes (right column)

Multiple public notes per file, visible to all users. Enter text in the 5-line text area and click Add Note. Each note card shows the author name in bold, the date, and the note content. Click the × button at the top-right corner of any card to delete it.

File Path Header

The full navigation path to the current file (Root / Serial / Step / File) is displayed in the header, so you always know which file the notes belong to.

Lists

The Lists page (accessible via the Lists tab) lets you organise HDF5 files into named collections. The page is split into two stacked sections:

Top Section — Your Lists

A scrollable container (up to 5 visible items) showing all lists you have access to. Each row displays:

  • Name: The list name (click to select it and view its files below)
  • Creator: The OS username of the list owner
  • Date: When the list was created
  • Public/Private toggle: Controls visibility to other users. Only the list owner can change this; for other users’ lists the toggle is read-only.
  • Delete button: Remove the list (owner only)

My Favs always appears as the first entry. It mirrors your favourited files and cannot be deleted or toggled.

Use the New List button to create a new list.

Bottom Section — Files in Selected List

Shows the files belonging to the currently selected list. Each file row displays its full cascade path (Root / Serial / Step / File) and two action buttons:

  • Show: Navigates the file selection bar to this file so you can immediately view its signals.
  • Remove: Removes the file from the list (with a confirmation prompt).

Use Add Current File to add whatever file is currently selected in the file selection bar.

Integration with File Lists Filter

Lists created here automatically appear in the File Lists combobox in the file selection bar. Selecting a list there filters the cascade dropdowns to only show files belonging to that list.

Favourites

Mark any file as a favourite by clicking the blue heart icon in the file selection bar. Favourites are stored per user and persist across sessions.

Favourited files can be quickly accessed by selecting My favs in the File Lists combobox, which prunes the cascade to show only the roots, serials, and steps that contain at least one favourited file.

Keyboard & Mouse Controls

Plot Interactions

  • Click and drag — Zoom into selected region (triggers adaptive data reload)
  • Shift + Click and drag — Pan across the plot
  • Double-click — Reset to full signal overview
  • House icon (toolbar) — Reset zoom to full overview

Toolbar Buttons

  • Zoom / Pan — Toggle drag mode
  • Zoom in / Zoom out — Incremental zoom
  • House icon — Reset zoom (full signal)
  • Camera icon — Download plot as PNG

Combobox Keyboard

  • Arrow Down / Arrow Up — Open dropdown or move highlight
  • Enter — Select highlighted item
  • Escape — Close dropdown
  • Type any text — Filter items by label
